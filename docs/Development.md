# Development

```console
$ npm install
$ npm run format
$ npm run build
$ npm run lint
$ npm run test
$ npm pack --dry-run
```

## Versioning

```console
$ npm \
  --no-git-tag-version \
  version [major|minor|patch|...]
```

## Distribution

```console
$ npm publish
```

## Node.js workspace with Docker Compose

```console
$ docker-compose run --rm workspace
```
